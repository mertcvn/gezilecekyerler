# İstanbul Nearby Places

İstanbul'da semt adı yazarak veya GPS ile en yakın **1000+** gezilecek yeri
anında listeleyen profesyonel web uygulaması.

## ✨ Özellikler
- 🗺️ Leaflet + Marker Cluster ile binlerce noktada akıcı performans
- 📍 Metin arama veya GPS ile anlık konum
- 🌓 Karanlık / Aydınlık mod
- 🏷️ 8 kategori, mesafe / alfabetik sıralama
- 🖼️ Detay modalı, Google Maps yol tarifi
- 📜 Arama geçmişi, tam ekran harita
- ♿ Erişilebilirlik ve klavye desteği

## 🚀 Kurulum
1. Repoyu klonlayın.
2. `node generate_places.js` komutu ile 1000+ mekanı oluşturun.
3. `index.html` dosyasını tarayıcıda açın.

## 📂 Veri Seti
`generate_places.js`, İstanbul'un tüm mahallelerini kullanarak
`data/places.json` üretir. Dilerseniz manuel düzenleyebilirsiniz.
